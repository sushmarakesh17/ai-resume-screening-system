# AI Resume Screening System

## Project Overview

The AI Resume Screening System is a Machine Learning and Natural Language Processing (NLP) based application that automates the resume screening process.

The system analyzes resume content, extracts relevant information, identifies skills, and helps in evaluating candidate suitability for a role. It reduces manual screening effort and improves the efficiency of the recruitment process.

## Objectives

- Automate resume analysis using Artificial Intelligence
- Extract meaningful information from resumes
- Match candidate skills with job requirements
- Improve the efficiency of candidate shortlisting

## Technologies Used

## Project Workflow

3. Text extraction and feature engineering
4. NLP-based text analysis
5. Machine Learning model training
6. Resume classification and prediction
7. Candidate evaluation
